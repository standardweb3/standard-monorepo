# Welcome to Standard Orderbook Kit

This is a UI library for Standard Exchange. Our library provides a set of reusable components designed to help you build consistent and high-quality user interfaces for your applications when you want to integrate orderbook exchange.

## Features

- **Reusable Components**: A collection of components that can be used across different projects.
- **Customizable**: Easily customize the components to fit your design needs.
- **Responsive Design**: Components are designed to be responsive and work on various screen sizes.
- **Accessibility**: Built with accessibility in mind to ensure a great experience for all users.

## Getting Started

To get started with the Standard Orderbook Kit, follow the instructions below:

1. **Installation**: Install the library using pnpm or yarn.
   ```sh
   pnpm install @standardweb3/orderbook-kit
   # or
   yarn add @standardweb3/orderbook-kit

2. **Usage**: Import the components you need into your project.
    ```javascript
    import { Button, Card } from '@standardweb3/orderbook-kit';

    const App = () => (
     <div>
       <Button>Click Me</Button>
       <Card title="Card Title">Card Content</Card>
     </div>
    );

    export default App;
    ```
