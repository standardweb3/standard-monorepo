# Welcome to Standard Orderbook Kit

This is a UI library for Standard Exchange. Our library provides a set of reusable components designed to help you build consistent and high-quality user interfaces for your applications when you want to integrate orderbook exchange.

## Features

- **Reusable Components**: A collection of components that can be used across different projects.
- **Customizable**: Easily customize the components to fit your design needs.
- **Responsive Design**: Components are designed to be responsive and work on various screen sizes.
- **Accessibility**: Built with accessibility in mind to ensure a great experience for all users.

## Design System Architecture

Our UI components follow Atomic Design principles, organizing elements from smallest to largest:

### Atoms

The smallest building blocks of our interface:

- Buttons
- Input fields
- Labels
- Icons
- Typography elements

### Molecules

Groups of atoms working together:

- Search bars (input + button)
- Form fields (label + input)
- Menu items (icon + text)
- Price displays (label + value)

### Organisms

Complex UI components combining molecules, this is the level where state logics from React hooks are included. Tests for hook state managements are done with:

- Order entry forms
- Price charts
- Order books
- Trading pairs selector
- Navigation bars
- Modals

### Templates(Features)

Page-level layouts that arrange organisms, each template takes part of a page and has a UX to test with:

- Tradingview Chart Feature
- Pair Search Feature
- Limit/Market Order Feature

### Pages

Specific instances of templates with real content, one page which is shown on browser once deployed:

- Spot trading page
- User portfolio page
- Market overview page
- Order history page

## Getting Started

To get started with the Standard Orderbook Kit, follow the instructions below:

1. **Installation**: Install the library using pnpm or yarn.

   ```sh
   pnpm install @standardweb3/orderbook-kit
   # or
   yarn add @standardweb3/orderbook-kit

   ```

2. **Usage**: Import the components you need into your project.

   ```javascript
   import { Button, Card } from '@standardweb3/orderbook-kit';

   const App = () => (
     <div>
       <Button>Click Me</Button>
       <Card title="Card Title">Card Content</Card>
     </div>
   );

   export default App;
   ```
